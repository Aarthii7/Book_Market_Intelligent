# 📚 Book Market Intelligence

## Overview

Book Market Intelligence is a data analytics project that explores publishing trends, reader engagement patterns, category performance, language distribution, and author influence using Google Books metadata. The project applies data cleaning, feature engineering, exploratory analysis, and statistical testing to transform raw book data into actionable business insights.

---

## Problem Statement

The publishing industry generates vast amounts of book-related data, making it difficult to identify the factors that drive reader engagement and market success. This project aims to analyze book metadata to uncover patterns in popularity, category performance, author influence, and publishing trends that can support data-driven decision-making.

---

## Dataset

**Source:** Google Books Metadata Dataset

**Initial Dataset:**

* 15,147 records
* 21 features

**Key Attributes:**

* Title
* Authors
* Categories
* Language
* Page Count
* Average Rating
* Ratings Count
* Publisher
* Publication Date
* List Price

## Outcome

The project successfully identified key drivers of reader engagement, evaluated category performance, analyzed publishing trends, and validated business assumptions through statistical testing. The findings provide actionable insights that can support publishers, authors, and market analysts in making informed decisions.

---

## Key Insights

* Fiction generated the highest reader engagement among all categories.
* English dominated the publishing landscape.
* Popularity differs significantly across book categories.
* Book length does not significantly influence popularity.
* Language and category are significantly associated.
* Established authors consistently achieved higher engagement levels.
* Correlation analysis revealed weak relationships between popularity and structural book characteristics.
